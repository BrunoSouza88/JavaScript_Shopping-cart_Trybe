<h1>Shopping-cart</h1>

<p>This is a project that implements a shopping cart functionality using JavaScript.</p>

<h2>Technologies Used</h2>

<p>The project utilizes the following technologies:</p>

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<h2>Prerequisites</h2>

<p>Before you begin, ensure that you meet the following requirements:</p>

<ul>
  <li>Have a modern web browser that supports JavaScript.</li>
</ul>

<h2>Installation</h2>

<p>Follow these steps to set up and run the project locally:</p>

<ol>
  <li>Clone the repository:</li>

  <pre><code>git clone https://github.com/BrunoSouza88/JavaScript_Shopping-cart_Trybe.git</code></pre>

  <li>Navigate to the project directory:</li>

  <pre><code>cd JavaScript_Shopping-cart_Trybe</code></pre>

  <li>Open the `index.html` file in a web browser.</li>
</ol>

<h2>Usage</h2>

<p>The project provides a shopping cart functionality where users can browse and add products to their cart. Users can view their cart, remove items, and proceed to checkout.</p>
